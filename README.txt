--------------READ ME--------------

Created by Alec Collins

The PyBank Program is located in the PyBank folder called PyBankCode.
It's used to see the profits of a Banks past years.

The PyPoll Program is located in the PyPoll folder called PyPollCode.
It's used to determine the winner of an election and see the amount of votes for each candidate.

When you run this code the results will be put in the Analysis folder for each respective program.

The CSV files are located in the Resources folder for each respective program.

In the Analysis folder I created a txt file that just says default in order for the Analysis folder to go through to github
